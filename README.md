# Library Management App

A *GUI* Library Management app made in python using tkinter and MySQL database.

### Features

- **Issue & Return Books**
    - Issue book
    - Return book
    - Show issued books
    
- **Manage Books**
    - Add new book
    - Update or Delete book
    - Search through books
    
- **Manage Members**
    - Add new member
    - Update or Delete members
    - Search through members
